## Description

`BDROM` is the main entry point for Blu-ray disc structure parsing. It discovers the BDMV directory layout, loads playlist (.mpls), stream (.m2ts), clip info (.clpi), and interleaved (.ssif) file metadata, and provides a `Scan()` method that recursively analyzes all discovered files. It also detects disc features like BD+, BD-Java, 3D, and D-BOX.

## Data Flow

```
path string ──► BDROM constructor
    ├── IFileSystem ──► Directory discovery (BDMV, PLAYLIST, STREAM, CLIPINF, etc.)
    ├── ITextEncoding ──► TSPlaylistFile, TSStreamClipFile parsing
    └── File enumeration ──► Dictionaries (PlaylistFiles, StreamFiles, StreamClipFiles, InterleavedFiles)

Scan()
    ├── StreamClipFiles.Values ──► TSStreamClipFile.Scan()
    ├── StreamFiles.Values ──► TSStreamFile.Scan(playlists, false)
    │   └── InterleavedFiles linked by SSIF name matching
    └── PlaylistFiles.Values ──► TSPlaylistFile.Scan(StreamFiles, StreamClipFiles)
        └── PlaylistFile.Initialize() ──► Detects 50Hz frame rates
```

## Side Effects

- Reads filesystem via `IFileSystem` (directory listings, file existence checks)
- Creates `TSPlaylistFile`, `TSStreamFile`, `TSStreamClipFile`, `TSInterleavedFile` instances
- Raises events (`StreamClipFileScanError`, `StreamFileScanError`, `PlaylistFileScanError`) on scan errors
- Throws `ArgumentNullException` if path is null/empty
- Throws `Exception` if BDMV structure cannot be located
